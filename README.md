# Fruit Loop

Ett terminalbaserat spel byggt i Python där spelaren rör sig på ett rutnät, samlar föremål och försöker få så många poäng som möjligt.

---

## Starta projektet

För att starta mitt projekt skriver man följande i terminalen, medan man står i projektets rotmapp.

```commandline
python -m src.game
```
## Hur man spelar

Du styr spelaren med WASD-tangenterna och rör dig på en karta. Målet är att samla föremål, få poäng och överleva så länge som möjligt.

Spelet innehåller väggar, frukter och specialföremål som nycklar, kistor och verktyg.

---

## Kontroller

W = Flytta upp  
A = Flytta vänster  
S = Flytta ner  
D = Flytta höger  
I = Visa inventory  
Q / X = Avsluta spelet  

---

## Spelets funktioner

- Rutnätsbaserad karta  
- Samla frukter och andra föremål för poäng  
- Inventory-system  
- Väggar som blockerar spelaren  
- Spade som kan bryta väggar  
- Nycklar och kistor 
- Poängen minskar med 1 per steg 

---

## Poängsystem
- Frukter: Ger +20 poäng styck
- Kista: Ger +100 poäng om du har en nyckel
- Steg: Varje steg drar av -1 poäng
